About this Project

The Mocs Arcade is a legacy project from a Computer Science student who wanted an outlet for creativity for other students. It is a Blue and Gold seven-foot arcade cabinet made and maintained by CompUTC, the college's programming club.

During my term as president of the club, we added wheels, arcade controls and a basic java launcher (seen above).

The launcher allows users to select a game using the arcade buttons. When a game is selected, the program runs a batch script that opens the game. This allows the launcher to support a wide variety of languages and programs.
